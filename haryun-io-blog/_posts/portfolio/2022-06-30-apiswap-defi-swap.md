## Project Overview

APISwap is a DeFi swap project delivering token swap functionality and related on-chain components (including wrapped-token work), alongside a web-based user interface.

## My Role

- Software Engineer (APIS / ecosystem project)

## Timeline

- **Jan 2022 - Jun 2022**

## Key Contributions

- Implemented swap smart contracts and validated behavior in a local test environment
- Built a web UI for swapping using React + Web3.js
- Set up and used Truffle/Ganache-based test flows to verify contract behavior
- Shipped the service and iterated based on usage feedback

## Tech Stack

- **Smart contracts**: Solidity
- **Testing**: Truffle, Ganache
- **Frontend**: React, Web3.js
